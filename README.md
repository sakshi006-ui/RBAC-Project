# RBAC System

Role Based Access Control (RBAC) System built using Spring Boot, React, TypeScript, JWT Authentication, and MySQL.

## Features

- User Registration
- User Login
- JWT Authentication
- Role Based Access Control
- USER Dashboard
- ADMIN Dashboard
- Protected Routes
- Logout Functionality

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios

## Setup Instructions

### Backend

```bash
cd backend/backend
./mvnw spring-boot:run
```

Backend runs on:

```
http://localhost:8080
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
